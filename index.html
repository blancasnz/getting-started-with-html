Basics of the Web and HTML

The Web

The basics of the web include The Web, HTML, URLS, HTTP, and Web Applications.

Major pieces of the web include You, the Browser, the Internet, HTTP, and Servers. When you go on a web browser (such as Chrome or Firefox), you connect to the Internet which connects to an HTTP (main protocol of the web) which then connects to a Server. The Server responds back with the correct page of what you searched for through HTTP which sends it back to your browser for you to view.

HTML

The worldwide web is made up of HTML documents. HTML stands for Hypertext Markup Language. It is the basis for everything on the web (what glues everything together). HTML is made up of text content (what you see), markup (what it looks like/arranged), and references to other documents (eg. images, videos, links to other pages).

HTML TAGS: b tag (bold), em tag (emphasis in italic), a tag (anchor tag used for making links by creating attributes), img tag (images), br tag (line breaks), and p tag (paragraphs).

Tags are either block elements (start on a new line/invisible box around them) or inline elements (just end a line/no invisible box around them).

Block elements include - p, div, form
Inline elements include - a, span, br, img, strong

Most tags have an opening tag and a closing tag (ex. p tag). Tags that don't require closing tags are called "void tags" (ex. br tag).

Main Points

What is a web page?
A web page is a text document written in language called HTML. Web browsers read these documents, and then interpret and display them.

How coding works
When programmers write text in a language that a computer can understand, the computer can then follow the instructions the programmer wrote.

Computers are stupid
Programmers need to write exactly the way a computer understands (also known as writing the correct "syntax".

Programmers Can't Remember Everything!

HTML tags are always contained within angled brackets. HTML elements refer to everything within a set of opening and closing tags. An attribute is a property of an HTML element (href attribute of anchor tag).
